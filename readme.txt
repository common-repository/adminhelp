== Description ==

Admin help menu creates an admin help menu where you can post html to provide help to your clients, such as video tutorials and other links.  Help menu is visible to users level 7 and higher.

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
